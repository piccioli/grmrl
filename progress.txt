## Codebase Patterns
- Alpine.js 3.15 installed via npm; initialized in `resources/js/app.js` with `import Alpine; window.Alpine = Alpine; Alpine.start()`
- `public/build` is gitignored — run `npm run build` on the host before testing UI (node is available outside Docker)
- Use `docker-compose exec app <command>` to run artisan/composer commands inside the container
- Filament requires `ext-intl` — add `libicu-dev` + `docker-php-ext-configure intl` + `docker-php-ext-install intl` to Dockerfile
- Filament 3.3 requires `symfony/console ^6|^7` but Laravel 13 locks it to v8. Install with `composer require filament/filament:'^3.3' --with-all-dependencies` to allow the downgrade to v7.4
- The User model must implement `Filament\Models\Contracts\FilamentUser` and define `canAccessPanel(Panel $panel): bool`
- `php artisan filament:install --panels` emits a harmless `xdg-open: not found` error in Docker (no browser) — installation succeeds anyway
- `/admin` returns 302 (redirect to login); follow with `-L` to get the 200 from the login page
