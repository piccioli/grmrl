## Codebase Patterns
- Artisan commands must be run via `docker compose exec app php artisan ...` (DB is in Docker, not accessible from host)
- Tests use `php artisan test` (no PHPStan, just Pint and Pest/PHPUnit)
- Migrations use `2026_06_25_XXXXXX_` prefix; add columns with `->after()` for clean ordering
- Use `Illuminate\Support\Facades\Http` for external HTTP calls (Laravel Http client is available); Pint enforces `concat_space` so use `' '` not `" "` for string concat
